🔥 Configuration Firebase (synchro)

📋 Étapes (2 minutes, gratuit)
1. Allez sur console.firebase.google.com
2. Cliquez Ajouter un projet (nom au choix)
3. Menu gauche : Build → Realtime Database
4. Cliquez Créer une base de données
5. Choisissez Démarrer en mode test
6. Copiez l'URL (ex: https://mon-projet-rtdb.firebaseio.com)
7. Collez-la ci-dessous et cliquez Enregistrer
💡 Une fois configuré, tous vos appareils partageront les données en temps réel avec le même code de synchro.
